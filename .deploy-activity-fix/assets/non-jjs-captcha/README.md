# JJS captcha assets

Положи сюда аватарки для капчи для тех, кто не играет в JJS.

- `1.png` ... `5.png` или `.jpg` / `.jpeg`: skillful персонажи
- `6.png` ... `10.png` или `.jpg` / `.jpeg`: non-skillful персонажи

Как бот собирает капчу:

- выбирает 1 случайный файл из `1..5`
- выбирает 1 случайный файл из `6..10`
- ставит 8 одинаковых skillful-картинок
- ставит 1 лишнюю non-skillful-картинку
- перемешивает позицию лишней картинки

Для работы капчи достаточно хотя бы одной картинки из `1..5` и хотя бы одной из `6..10`, но лучше заполнить все 10 слотов.
